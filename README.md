# GameTrust - 游戏服务担保交易平台

## 项目简介

GameTrust 是一个专注于游戏代肝、账号交易、道具交易的官方担保交易与玩家社群平台，通过"社群交流 + 官方担保"模式解决游戏服务交易中的信任问题。

## 技术栈

- **前端框架**: Vue 3 + Vite
- **UI组件库**: Vant 4 (移动端UI组件库)
- **状态管理**: Pinia
- **路由管理**: Vue Router 4
- **样式预处理**: Sass
- **HTTP客户端**: Axios

## 开发环境要求

- Node.js >= 16.0.0
- npm >= 8.0.0 或 yarn >= 1.22.0

## 安装与运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

4. 预览生产版本：
```bash
npm run preview
```
